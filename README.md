# AIML-Task-9-Credit-Card-Fraud
# Credit Card Fraud Detection using Random Forest
## Dataset
Kaggle Credit Card Fraud Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- SMOTE
- Random Forest

## Workflow
1. Dataset loading and analysis
2. Handling class imbalance using SMOTE
3. Logistic Regression baseline model
4. Random Forest classifier training
5. Evaluation using precision, recall and F1-score
6. Feature importance visualization
7. Model saved using joblib

## Deliverables
- Jupyter Notebook
- Feature importance plot
- Saved model (.pkl)

## Outcome
Learned ensemble learning and handling imbalanced datasets.
